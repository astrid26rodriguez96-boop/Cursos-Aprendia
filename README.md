📚 Aprendía – Plataforma de Cursos en Línea

Aprendía es un proyecto de base de datos desarrollado para gestionar una plataforma de cursos en línea. Permite administrar estudiantes, instructores, cursos e inscripciones, además de analizar la información mediante consultas SQL para apoyar la toma de decisiones.

🚀 Descripción del Proyecto

Este proyecto fue creado con el objetivo de simular el funcionamiento de una plataforma educativa, permitiendo almacenar, organizar y analizar datos relacionados con el aprendizaje en línea.

A través de consultas SQL, se responden preguntas clave del negocio, como el comportamiento de los estudiantes, el rendimiento de los cursos y los patrones de inscripción.

🧩 Funcionalidades
📌 Gestión de estudiantes e instructores
📌 Creación y administración de cursos
📌 Inscripción de estudiantes
📌 Organización en módulos y lecciones
📌 Seguimiento del progreso
📌 Análisis de datos con SQL
🗄️ Base de Datos

El proyecto utiliza una base de datos relacional en MySQL, estructurada con tablas como:

Estudiantes
Instructores
Cursos
Inscripciones
Módulos
Lecciones
💻 Tecnologías Utilizadas
MySQL
SQL (JOIN, GROUP BY, COUNT, etc.)
MySQL Workbench
Git y GitHub
📊 Preguntas de Negocio

Algunas de las preguntas que se resolvieron con SQL:

📈 ¿Qué cursos tienen más estudiantes inscritos?
👨‍🎓 ¿Qué estudiantes están inscritos en más cursos?
📅 ¿En qué mes hay más inscripciones?
✅ ¿Cuántos estudiantes completan los cursos?
📈 Resultados

Gracias a las consultas realizadas, se logró:

Identificar cursos más populares
Analizar el comportamiento de los estudiantes
Evaluar el rendimiento de instructores
Obtener información clave para la toma de decisiones
